# 🚀 Plano de Carreira

Este repositório contém meu plano de estudos para três carreiras promissoras e com alta demanda no mercado:  
**Análise de Dados**, **Suporte Técnico/Help Desk** e **Produção de Conteúdo/Copywriting**.

---

## 📊 Análise de Dados
**Objetivo:** aprender a coletar, organizar e interpretar dados para gerar insights.

### Skills
- Estatística básica
- Excel / Google Sheets
- Power BI
- SQL
- Python (pandas, numpy, matplotlib)
- Dashboards e relatórios

### Cursos recomendados
- [Google Data Analytics – Coursera](https://www.coursera.org/professional-certificates/google-data-analytics)
- [Python para Análise de Dados – Alura](https://www.alura.com.br)
- [Power BI – Udemy](https://www.udemy.com)

### Roadmap
- Fundamentos: 20h  
- Ferramentas: 40h  
- Linguagens: 60h  
- Projetos práticos: 30h  

**Total:** 150 horas  
- Estudando 2h/dia → 75 dias (~2,5 meses)  
- Estudando 4h/dia → 38 dias (~1,5 mês)

## 🗺️ Mapa de Skills
- **Dados + Conteúdo:** usar análise de dados para criar conteúdo estratégico.  
- **Suporte + Dados:** aplicar lógica de TI para organizar informações.  
- **Conteúdo + Suporte:** comunicar soluções técnicas de forma clara e persuasiva.

---

## 📌 Ordem sugerida
1. **Análise de Dados** → base técnica e lógica.  
2. **Suporte Técnico** → empregabilidade rápida e fundamentos de TI.  
3. **Copywriting** → habilidades criativas e comunicação estratégica.

---

## ⏱️ Cronograma de Estudos
- **2h/dia:** progresso consistente, conclusão em 2–3 meses por carreira.  
- **4h/dia:** aceleração, conclusão em 1–1,5 mês por carreira.

**Exemplo semanal (2h/dia):**
- Segunda a Sexta: 1h teoria + 1h prática  
- Sábado: revisão + mini projeto (2h)  
- Domingo: descanso ou leitura leve

---

## 📌 Reflexão
Com dedicação de **2 a 4 horas por dia**, é possível concluir cada carreira em **1 a 3 meses**.  
O segredo é constância: mesmo 2h/dia já traz resultados sólidos.

> *“Qual o plano de carreira em estudos pode me ajudar a fazer, estudando 2 a 4 horas por dia, e em quantas horas consigo ter êxito?”*

Este documento responde a essa pergunta e organiza o caminho para alcançar êxito nas três áreas escolhidas.
